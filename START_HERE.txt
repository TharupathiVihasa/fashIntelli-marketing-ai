START HERE

This package is designed to run from the BACKEND only for normal use.
The backend serves the completed GUI directly.

Run:
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000

Then open:
http://localhost:8000

Useful routes:
- / = GUI
- /api = API info
- /docs = FastAPI docs
- /api/health = backend health check

If you used an older version before, delete this file once first:
backend/storage/fashintelli.db


Analysts can now upload replacement datasets from the Datasets tab and optionally retrain immediately from the frontend.
